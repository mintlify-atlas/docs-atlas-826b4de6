The GRVT JavaScript Client SDK is written in TypeScript and provides comprehensive type definitions for all API methods, WebSocket streams, and data structures. This guide shows you how to use TypeScript effectively with the SDK.

## Installation with TypeScript

The SDK includes TypeScript definitions out of the box:

```bash
npm install @grvt/client
# or
yarn add @grvt/client
```

<Note>
No additional `@types` packages are needed. TypeScript definitions are bundled with the SDK.
</Note>

## Configuration Types

### SDK Configuration

```typescript
import { IConfig } from '@grvt/client';

const config: IConfig = {
  host: 'https://api.grvt.io',
  version: 'v1' // optional, defaults to 'v1'
};

// Type-safe validation
import { validConfig } from '@grvt/client';

const validatedConfig = validConfig(config);
// Returns: { host: string, version: `v${number}` }
```

## MDG Types

### Request and Response Types

All MDG methods have strongly typed request and response interfaces:

```typescript
import { 
  MDG,
  IApiGetInstrumentRequest,
  IApiGetInstrumentResponse,
  IApiTickerRequest,
  IApiTickerResponse,
  IApiOrderbookLevelsRequest,
  IApiOrderbookLevelsResponse
} from '@grvt/client';

const mdg = new MDG(config);

// Type-safe instrument query
const instrumentRequest: IApiGetInstrumentRequest = {
  instrument: 'BTC_USDT_Perp'
};

const instrumentResponse: IApiGetInstrumentResponse = 
  await mdg.instrument(instrumentRequest);

// TypeScript knows the shape of the response
console.log(instrumentResponse.result.instrument);
console.log(instrumentResponse.result.tick_size);

// Type-safe ticker request
const tickerRequest: IApiTickerRequest = {
  instrument: 'BTC_USDT_Perp'
};

const tickerResponse: IApiTickerResponse = 
  await mdg.ticker(tickerRequest);

// Full autocomplete and type checking
const lastPrice = tickerResponse.result.last_price;
const markPrice = tickerResponse.result.mark_price;
const volume24h = tickerResponse.result.volume_24h;
```

### WebSocket Request Types

```typescript
import {
  WS,
  EStream,
  IWSCandleRequest,
  IWSTickerRequest,
  IWSBookRequest,
  IWSTdgOrderRequest,
  IWSTdgPositionRequest,
  IWSTdgFillRequest
} from '@grvt/client';

const ws = new WS({ url: 'wss://ws.grvt.io' });

// Type-safe candlestick subscription
const candleRequest: IWSCandleRequest = {
  stream: EStream.CANDLE,
  params: {
    instrument: 'BTC_USDT_Perp',
    interval: '1H', // Type enforced
    type: 'MARK' // Type enforced: 'MARK' | 'INDEX' | 'LAST'
  },
  onData: (candle) => {
    // candle is strongly typed
    console.log(candle.open, candle.high, candle.low, candle.close);
  },
  onError: (error) => {
    console.error(error.message);
  }
};

ws.subscribe(candleRequest);

// Type-safe order subscription
const orderRequest: IWSTdgOrderRequest = {
  stream: EStream.ORDER,
  params: {
    sub_account_id: 'sub_account_123',
    instrument: 'BTC_USDT_Perp'
  },
  onData: (order) => {
    // order is fully typed
    console.log({
      order_id: order.order_id,
      status: order.status,
      filled_size: order.filled_size
    });
  }
};

ws.subscribe(orderRequest);
```

## Advanced TypeScript Patterns

### Type Guards

```typescript
import { IWSOrderFeedDataV1, IWSFillFeedDataV1 } from '@grvt/client';

// Create type guards for runtime type checking
function isOrderUpdate(
  data: any
): data is Required<IWSOrderFeedDataV1>['feed'] {
  return 'order_id' in data && 'status' in data;
}

function isFillUpdate(
  data: any
): data is Required<IWSFillFeedDataV1>['feed'] {
  return 'order_id' in data && 'price' in data && 'fee' in data;
}

// Use type guards
function processUpdate(data: any) {
  if (isOrderUpdate(data)) {
    console.log('Order:', data.order_id, data.status);
  } else if (isFillUpdate(data)) {
    console.log('Fill:', data.order_id, data.price, data.size);
  }
}
```

### Generic Wrapper Functions

```typescript
import { MDG, TDG } from '@grvt/client';

// Create type-safe wrapper functions
async function withRetry<T>(
  fn: () => Promise<T>,
  maxRetries: number = 3
): Promise<T> {
  let lastError: Error;
  
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await fn();
    } catch (error) {
      lastError = error as Error;
      await new Promise(resolve => setTimeout(resolve, 1000 * (i + 1)));
    }
  }
  
  throw lastError!;
}

// Usage with full type safety
const mdg = new MDG(config);

const ticker = await withRetry(() => 
  mdg.ticker({ instrument: 'BTC_USDT_Perp' })
);
// ticker is typed as IApiTickerResponse
```

### Custom Type Definitions

```typescript
import { IApiCreateOrderRequest } from '@grvt/client';

// Extend SDK types for your application
interface MyOrderRequest extends IApiCreateOrderRequest {
  metadata?: {
    strategy: string;
    user_id: string;
  };
}

interface OrderWithMetadata {
  request: MyOrderRequest;
  submitted_at: number;
  response?: any;
}

// Use custom types
const orders: OrderWithMetadata[] = [];

function submitOrder(request: MyOrderRequest) {
  orders.push({
    request,
    submitted_at: Date.now()
  });
  
  return tdg.createOrder(request);
}
```

## Type-Safe Error Handling

```typescript
import { WSError } from '@grvt/client';

// Type guard for WebSocket errors
function isWSError(error: unknown): error is WSError {
  return error instanceof WSError;
}

// Type-safe error handling
try {
  const ticker = await mdg.ticker({ instrument: 'BTC_USDT_Perp' });
  console.log(ticker.result);
} catch (error) {
  if (isWSError(error)) {
    console.error('WebSocket error:', {
      code: error.code,
      status: error.status,
      message: error.message
    });
  } else if (error instanceof Error) {
    console.error('General error:', error.message);
  } else {
    console.error('Unknown error:', error);
  }
}
```

## Best Practices

<AccordionGroup>
  <Accordion title="Enable Strict Type Checking">
    Configure your `tsconfig.json` for maximum type safety:
    
    ```json
    {
      "compilerOptions": {
        "strict": true,
        "noImplicitAny": true,
        "strictNullChecks": true,
        "strictFunctionTypes": true,
        "noUnusedLocals": true,
        "noUnusedParameters": true
      }
    }
    ```
  </Accordion>
  
  <Accordion title="Use Type Inference">
    Let TypeScript infer types when possible:
    
    ```typescript
    // Good: Type inferred from SDK method
    const ticker = await mdg.ticker({ instrument: 'BTC_USDT_Perp' });
    
    // Unnecessary: Explicitly typing when not needed
    const ticker: IApiTickerResponse = await mdg.ticker({ ... });
    ```
  </Accordion>
  
  <Accordion title="Create Domain-Specific Types">
    Build application types on top of SDK types:
    
    ```typescript
    import { IApiPositionsResponse } from '@grvt/client';
    
    type PositionWithMetrics = IApiPositionsResponse['result'][0] & {
      roi: number;
      duration: number;
    };
    ```
  </Accordion>
</AccordionGroup>
