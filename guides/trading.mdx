The TDG (Trading Data Gateway) class provides comprehensive trading functionality including order management, position tracking, account operations, and transfer management. This guide covers essential trading workflows.

## Initialization

Initialize the TDG client with your configuration:

```typescript
import { TDG } from '@grvt/client';

const tdg = new TDG({
  host: 'https://api.grvt.io',
  version: 'v1' // optional, defaults to 'v1'
});
```

<Note>
All trading operations require authentication. Ensure your requests include proper authentication headers.
</Note>

## Order Management

### Creating Orders

<Steps>
  <Step title="Pre-order Check">
    Before placing an order, validate it with the pre-order check:
    
    ```typescript
    const preCheck = await tdg.preOrderCheck({
      sub_account_id: 'sub_account_123',
      instrument: 'BTC_USDT_Perp',
      size: '1.5',
      limit_price: '45000',
      side: 'BUY',
      time_in_force: 'GTC'
    });
    
    console.log(preCheck.result);
    // Validates margin requirements and position limits
    ```
  </Step>
  
  <Step title="Create Single Order">
    Place a single order:
    
    ```typescript
    const order = await tdg.createOrder({
      sub_account_id: 'sub_account_123',
      instrument: 'BTC_USDT_Perp',
      size: '1.5',
      limit_price: '45000',
      side: 'BUY',
      time_in_force: 'GTC',
      order_type: 'LIMIT'
    });
    
    console.log(order.result);
    // Returns order_id and order details
    ```
  </Step>
  
  <Step title="Create Bulk Orders">
    Place multiple orders in a single request:
    
    ```typescript
    const bulkOrders = await tdg.createBulkOrders({
      sub_account_id: 'sub_account_123',
      orders: [
        {
          instrument: 'BTC_USDT_Perp',
          size: '1.0',
          limit_price: '45000',
          side: 'BUY',
          time_in_force: 'GTC',
          order_type: 'LIMIT'
        },
        {
          instrument: 'ETH_USDT_Perp',
          size: '10.0',
          limit_price: '3000',
          side: 'BUY',
          time_in_force: 'GTC',
          order_type: 'LIMIT'
        }
      ]
    });
    
    console.log(bulkOrders.result);
    ```
  </Step>
</Steps>

### Canceling Orders

#### Cancel a Single Order

```typescript
const cancelResult = await tdg.cancelOrder({
  sub_account_id: 'sub_account_123',
  order_id: 'order_456'
});

console.log(cancelResult.result);
```

#### Cancel All Orders

Cancel all open orders, optionally filtered by instrument:

```typescript
// Cancel all orders for a specific instrument
const cancelAll = await tdg.cancelAllOrders({
  sub_account_id: 'sub_account_123',
  instrument: 'BTC_USDT_Perp'
});

// Cancel all orders across all instruments
const cancelAllInstruments = await tdg.cancelAllOrders({
  sub_account_id: 'sub_account_123'
});

console.log(cancelAll.result);
```

## Position Management


## Error Handling

Implement comprehensive error handling for trading operations:

```typescript
try {
  const order = await tdg.createOrder({
    sub_account_id: 'sub_account_123',
    instrument: 'BTC_USDT_Perp',
    size: '1.5',
    limit_price: '45000',
    side: 'BUY',
    time_in_force: 'GTC',
    order_type: 'LIMIT'
  });
  console.log('Order created:', order.result);
} catch (error) {
  console.error('Order failed:', error.message);
  
  // Handle specific error cases
  if (error.code === 'INSUFFICIENT_MARGIN') {
    console.error('Not enough margin to place order');
  } else if (error.code === 'INVALID_PRICE') {
    console.error('Price is outside acceptable bounds');
  }
}
```
