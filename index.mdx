## Quick Start

<Steps>
  <Step title="Install the package">
    Install the GRVT client using your preferred package manager.
    
    <CodeGroup>
    ```bash npm
    npm install @grvt/client
    ```
    ```bash yarn
    yarn add @grvt/client
    ```
    ```bash pnpm
    pnpm add @grvt/client
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Initialize the client">
    Import and configure the Market Data Gateway or Trading Data Gateway.
    
    ```typescript
    import GRVT from '@grvt/client'
    
    // Initialize Market Data Gateway
    const mdg = new GRVT.MDG({
      host: 'https://market-data.grvt.io',
      version: 'v1'
    })
    
    // Initialize Trading Data Gateway
    const tdg = new GRVT.TDG({
      host: 'https://trading.grvt.io',
      version: 'v1'
    })
    ```
  </Step>
  
  <Step title="Start making requests">
    Use the SDK to interact with GRVT APIs.
    
    ```typescript
    // Fetch instrument data
    const instruments = await mdg.allInstruments({})
    
    // Get market ticker
    const ticker = await mdg.ticker({
      instrument: 'BTC-USD-PERP'
    })
    
    // Create an order
    const order = await tdg.createOrder({
      instrument: 'BTC-USD-PERP',
      size: '0.1',
      // ... additional order parameters
    })
    ```
  </Step>
</Steps>
