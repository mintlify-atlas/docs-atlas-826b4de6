Request types used across the GRVT SDK for various API operations.

## WebSocket Request Types

### IWSCandleRequest

Request interface for candlestick data subscription.

```typescript
interface IWSCandleRequest {
  stream: `${EStream.CANDLE}`
  params: {
    interval: `${ECandlestickInterval}`
    type: `${ECandlestickType}`
    instrument: string
  }
  onData?: TMessageHandler<Required<IWSCandlestickFeedDataV1>['feed']>
  onError?: (error: Error) => void
}
```

<ParamField path="stream" type="'candle'" required>
  Stream identifier for candlestick data
</ParamField>

<ParamField path="params.interval" type="ECandlestickInterval" required>
  Candlestick interval (e.g., `CI_1_M`, `CI_5_M`, `CI_1_H`)
</ParamField>

<ParamField path="params.type" type="ECandlestickType" required>
  Price type: `TRADE`, `MARK`, `INDEX`, or `MID`
</ParamField>

<ParamField path="params.instrument" type="string" required>
  Instrument identifier
</ParamField>

<ParamField path="onData" type="TMessageHandler">
  Callback function for candlestick updates
</ParamField>

<ParamField path="onError" type="(error: Error) => void">
  Error handler callback
</ParamField>

### IWSBookRequest

Request interface for orderbook subscription.

```typescript
interface IWSBookRequest {
  stream: `${EStream.ORDERBOOK_DELTA}` | `${EStream.ORDERBOOK_SNAP}` | `${EStream.RPI_BOOK_DELTA}` | `${EStream.RPI_BOOK_SNAP}`
  params: IWSOrderbookLevelsFeedSelectorV1
  onData?: TMessageHandler<Required<IWSOrderbookLevelsFeedDataV1>['feed']>
  onError?: (error: Error) => void
}
```

<ParamField path="stream" type="string" required>
  Stream type: `rpi.book.d`, `rpi.book.s`, or deprecated `book.d`, `book.s`
</ParamField>

<ParamField path="params" type="IWSOrderbookLevelsFeedSelectorV1" required>
  Orderbook feed selector with instrument, rate, and depth parameters
</ParamField>

## Union Types

### TWSRequest

Union of all WebSocket request types.

```typescript
type TWSRequest =
  | IWSCandleRequest
  | IWSBookRequest
  | IWSMiniRequest
  | IWSTickerRequest
  | IWSTradeRequest
  | IWSTdgOrderGroupRequest
  | IWSTdgOrderRequest
  | IWSTdgOrderStateRequest
  | IWSTdgPositionRequest
  | IWSTdgFillRequest
  | IWSTdgDepositRequest
  | IWSTdgTransferRequest
  | IWSTdgWithDrawalRequest
```

This union type is used by the `subscribe()` and `subscribes()` methods to accept any valid stream request.
