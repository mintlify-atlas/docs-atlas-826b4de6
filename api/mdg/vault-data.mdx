Vault data methods provide access to vault performance metrics, risk data, positions, and historical information for vaults on the Gravity platform.

## vaultAssetExposure

Get vault asset exposure over a specified time period.

```typescript
vaultAssetExposure(payload: IApiTimedVaultAssetExposureRequest, config?: AxiosRequestConfig): Promise<IApiTimedAssetExposureResponse>
```

### Parameters

<ParamField path="payload" type="IApiTimedVaultAssetExposureRequest" required>
  <Expandable title="properties">
    <ParamField path="vault_id" type="string" required>
      The vault ID to query
    </ParamField>
    
    <ParamField path="start_time" type="string">
      Start time in unix nanoseconds
    </ParamField>
    
    <ParamField path="end_time" type="string">
      End time in unix nanoseconds
    </ParamField>
  </Expandable>
</ParamField>

<ParamField path="config" type="AxiosRequestConfig">
  Optional Axios request configuration
</ParamField>

### Response

<ResponseField name="result" type="IApiTimedAssetExposureResponse">
  Contains vault asset exposure data over time
</ResponseField>

### Example

```typescript
const exposure = await mdg.vaultAssetExposure({
  vault_id: 'vault_123',
  start_time: '1634000000000000000',
  end_time: '1634086400000000000'
});

console.log('Asset exposure:', exposure.result);
```

## vaultSummaryHistory

Get historical summary data for a vault.

```typescript
vaultSummaryHistory(payload: IApiQueryVaultSummaryHistoryRequest, config?: AxiosRequestConfig): Promise<IApiQueryVaultSummaryHistoryResponse>
```

### Parameters

<ParamField path="payload" type="IApiQueryVaultSummaryHistoryRequest" required>
  <Expandable title="properties">
    <ParamField path="vault_id" type="string" required>
      The vault ID to query
    </ParamField>
    
    <ParamField path="start_time" type="string">
      Start time in unix nanoseconds
    </ParamField>
    
    <ParamField path="end_time" type="string">
      End time in unix nanoseconds
    </ParamField>
    
    <ParamField path="limit" type="number">
      Maximum number of records to return
    </ParamField>
    
    <ParamField path="cursor" type="string">
      Pagination cursor
    </ParamField>
  </Expandable>
</ParamField>

<ParamField path="config" type="AxiosRequestConfig">
  Optional Axios request configuration
</ParamField>

### Response

<ResponseField name="result" type="IApiQueryVaultSummaryHistoryResponse">
  <Expandable title="properties">
    <ResponseField name="result" type="VaultSummary[]">
      Array of historical vault summaries
    </ResponseField>
    
    <ResponseField name="next" type="string">
      Pagination cursor for the next page
    </ResponseField>
  </Expandable>
</ResponseField>

### Example

```typescript
const history = await mdg.vaultSummaryHistory({
  vault_id: 'vault_123',
  limit: 100
});

console.log(`Retrieved ${history.result.length} historical summaries`);

// Fetch next page if available
if (history.next) {
  const nextPage = await mdg.vaultSummaryHistory({
    vault_id: 'vault_123',
    cursor: history.next
  });
}
```

## Common Use Cases

### Monitor vault performance

```typescript
// Get current performance metrics
const performance = await mdg.vaultPerformance({
  vault_i_ds: ['vault_123'],
  use_annualized_returns: true
});

// Get performance trend over the last 30 days
const thirtyDaysAgo = String(Date.now() * 1000000 - 30 * 24 * 60 * 60 * 1000000000);
const now = String(Date.now() * 1000000);

const trend = await mdg.vaultPerformanceTrend({
  vault_id: 'vault_123',
  start_time: thirtyDaysAgo,
  end_time: now,
  interval: 'DAY'
});
```

### Analyze vault risk

```typescript
// Get risk metrics for multiple vaults
const riskData = await mdg.vaultRiskMetric({
  vault_i_ds: ['vault_123', 'vault_456', 'vault_789']
});

// Sort by risk level
const sortedByRisk = riskData.result.sort((a, b) => 
  parseFloat(a.risk_score) - parseFloat(b.risk_score)
);
```

### Track vault positions

```typescript
// Get current position snapshot
const currentPosition = await mdg.snapVaultPosition({
  vault_id: 'vault_123'
});

// Get asset exposure over time
const exposure = await mdg.vaultAssetExposure({
  vault_id: 'vault_123',
  start_time: thirtyDaysAgo,
  end_time: now
});
```
