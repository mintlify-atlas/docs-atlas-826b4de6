## Installation

First, install the GRVT JavaScript Client SDK:

```bash
npm install @grvt/client
```

## Initializing the Client

The SDK provides three main classes:
- **MDG** (Market Data Gateway): Access public market data
- **TDG** (Trading Data Gateway): Execute trades and manage your account
- **WS** (WebSocket): Real-time data streams

### Basic Configuration

<CodeGroup>
```typescript TypeScript
import { MDG, TDG, WS } from '@grvt/client';

const config = {
  host: 'https://api.grvt.io',
  version: 'v1'
};

// Initialize market data client
const mdg = new MDG(config);

// Initialize trading client (requires authentication)
const tdg = new TDG(config);

// Initialize WebSocket for real-time data
const ws = new WS({
  url: 'wss://ws.grvt.io',
  reconnectStrategy: (retries) => {
    // Exponential backoff: 1s, 2s, 4s, 8s, max 30s
    return Math.min(1000 * Math.pow(2, retries), 30000);
  }
});
```


## Getting Market Data

### Fetch Instrument Information

```typescript
// Get single instrument details
const instrument = await mdg.instrument({
  instrument: 'BTC_USDT_Perp'
});

console.log('Instrument:', instrument.result);
```

### Get Current Ticker

```typescript
// Get ticker for BTC perpetual
const ticker = await mdg.ticker({
  instrument: 'BTC_USDT_Perp'
});

console.log('Last price:', ticker.result.last_price);
console.log('24h volume:', ticker.result.volume_24h);
console.log('24h change:', ticker.result.price_change_24h);
```

### Fetch Order Book

```typescript
// Get order book with depth of 20 levels
const orderBook = await mdg.orderBook({
  instrument: 'BTC_USDT_Perp',
  depth: 20
});

console.log('Best bid:', orderBook.result.bids[0]);
console.log('Best ask:', orderBook.result.asks[0]);
```

## Placing Your First Trade

<Note>
Trading requires authentication. Make sure you've configured your API credentials before executing trades.
</Note>

### Check Account Balance

```typescript
// Get sub-account summary
const accountSummary = await tdg.subAccountSummary({
  sub_account_id: 'YOUR_SUB_ACCOUNT_ID'
});

console.log('Total equity:', accountSummary.result.total_equity);
console.log('Available balance:', accountSummary.result.total_available_balance);
```

### Create a Limit Order

```typescript
// Place a limit buy order for BTC
const order = await tdg.createOrder({
  sub_account_id: 'YOUR_SUB_ACCOUNT_ID',
  instrument: 'BTC_USDT_Perp',
  is_market: false,
  size: '0.1', // Buy 0.1 BTC
  limit_price: '50000.00', // At $50,000
  time_in_force: 'GTC' // Good till cancelled
});

console.log('Order ID:', order.result.order_id);
console.log('Status:', order.result.metadata.status);
```

### Create a Market Order

```typescript
// Place a market sell order
const marketOrder = await tdg.createOrder({
  sub_account_id: 'YOUR_SUB_ACCOUNT_ID',
  instrument: 'BTC_USDT_Perp',
  is_market: true,
  size: '-0.1', // Sell 0.1 BTC (negative size)
  time_in_force: 'IOC' // Immediate or cancel
});

console.log('Filled price:', marketOrder.result.metadata.avg_fill_price);
```

## Monitoring Positions

```typescript
// Get all open positions
const positions = await tdg.positions({
  sub_account_id: 'YOUR_SUB_ACCOUNT_ID'
});

for (const position of positions.result) {
  console.log('Instrument:', position.instrument);
  console.log('Size:', position.size);
  console.log('Entry price:', position.avg_entry_price);
  console.log('Unrealized PnL:', position.unrealized_pnl);
}
```

## Error Handling

Always wrap API calls in try-catch blocks:

```typescript
try {
  const ticker = await mdg.ticker({
    instrument: 'BTC_USDT_Perp'
  });
  console.log('Price:', ticker.result.last_price);
} catch (error) {
  if (error.response) {
    // API returned an error response
    console.error('API Error:', error.response.data);
  } else if (error.request) {
    // Request was made but no response received
    console.error('Network Error:', error.message);
  } else {
    // Something else went wrong
    console.error('Error:', error.message);
  }
}
```

## Complete Example

Here's a complete example that demonstrates fetching market data and placing a trade:

```typescript
import { MDG, TDG } from '@grvt/client';

const config = {
  host: 'https://api.grvt.io',
  version: 'v1'
};

const mdg = new MDG(config);
const tdg = new TDG(config);

async function tradingWorkflow() {
  try {
    // 1. Get current market price
    const ticker = await mdg.ticker({
      instrument: 'BTC_USDT_Perp'
    });
    const currentPrice = parseFloat(ticker.result.last_price);
    console.log('Current BTC price:', currentPrice);

    // 2. Check account balance
    const account = await tdg.subAccountSummary({
      sub_account_id: 'YOUR_SUB_ACCOUNT_ID'
    });
    console.log('Available balance:', account.result.total_available_balance);

    // 3. Calculate order size based on available balance
    const orderSize = '0.01'; // Conservative size

    // 4. Place limit order 1% below current price
    const limitPrice = (currentPrice * 0.99).toFixed(2);
    const order = await tdg.createOrder({
      sub_account_id: 'YOUR_SUB_ACCOUNT_ID',
      instrument: 'BTC_USDT_Perp',
      is_market: false,
      size: orderSize,
      limit_price: limitPrice,
      time_in_force: 'GTC'
    });

    console.log('Order placed successfully!');
    console.log('Order ID:', order.result.order_id);
    console.log('Limit price:', limitPrice);

    // 5. Monitor open orders
    const openOrders = await tdg.openOrders({
      sub_account_id: 'YOUR_SUB_ACCOUNT_ID',
      instrument: 'BTC_USDT_Perp'
    });
    console.log('Total open orders:', openOrders.result.length);

  } catch (error) {
    console.error('Trading workflow error:', error.message);
  }
}

tradingWorkflow();
```
